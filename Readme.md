# Mood Journal CLI

A beginner-friendly Python project that lets users log daily moods and view simple analytics.

## Features
- Add a new journal entry (date, mood, note)
- View past entries
- Calculate the average mood for the week

## Skills Practiced
- File handling
- Loops and conditional logic
- CSV storage and basic data analysis

## How to Run
1. Download `mood_journal.py`.
2. Run in a terminal:
   ```bash
   python mood_journal.py
